# Malik Masood — Software Engineer Portfolio

A personal portfolio website showcasing projects, skills, experience, and certifications. Built with vanilla HTML, CSS, and JavaScript.

## Project Structure

```
Portfolio_Project/
│
├── index.html              # Main landing page
│
├── css/
│   ├── style.css           # Main styles, design tokens, animations
│   └── responsive.css      # Responsive breakpoints
│
├── js/
│   └── script.js           # Interactivity, animations, form validation
│
├── images/
│   ├── Remote App Launcher.png
│   ├── Interactive portfolio website.png
│   └── Esport Production.mp4
│
├── assets/
│   └── (cv.pdf)            # Resume / CV (add your own)
│
└── README.md               # This file
```

## Sections

1. **Hero** — Introduction with animated background and typing effect
2. **About Me** — Bio, highlights, and quick stats
3. **Skills** — Technical skills with animated progress bars and tools grid
4. **Projects** — Filterable project cards with images and tech stacks
5. **Education Timeline** — Education & work experience with tabbed timeline
6. **Certifications** — Professional certifications and interests
7. **Contact Form** — Functional contact form with validation
8. **Footer** — Social links and navigation

## Features

- **Responsive Design** — Works on desktop, tablet, and mobile
- **Dark / Light Theme Toggle** — Switch between color modes
- **Animated Hero Background** — Subtle moving patterns and gradients
- **Smooth Counter Animations** — Stat numbers count up on scroll
- **Project Filtering** — Filter projects by category
- **Timeline Tabs** — Switch between Education and Experience
- **Form Validation** — Real-time validation with success/error states
- **Accessibility** — Skip links, ARIA labels, semantic HTML

## How to Run

Open `index.html` directly in any modern browser:

```
# Or serve locally (recommended)
npx serve .
# or
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Tech Stack

- HTML5 (semantic, accessible)
- CSS3 (custom properties, flexbox, grid, animations)
- Vanilla JavaScript (no frameworks)
- Font Awesome 6 (icons)
- Google Fonts (Playfair Display, Inter, JetBrains Mono)
